# Archivist

Welcome to the official documentation for Archivist - a free, locally-run retrieval augmented generation (RAG) application.

## What is Archivist?

Archivist is a desktop application that enables you to chat with your documents using AI. Built with privacy in mind, it runs entirely on your local machine.

## Key Features

- **Local-first**: All processing happens on your machine
- **Free to use**: Full functionality at no cost
- **Premium models**: Unlock the ability to swap AI models with a paid upgrade
- **Multiple document formats**: Support for various file types via IBM Docling
- **Vector search**: Powered by Chroma and Nomic embeddings
- **Cross-platform**: Available on Windows via Microsoft Store
